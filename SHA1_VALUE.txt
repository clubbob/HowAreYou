# Firebase Console에 등록할 SHA-1 지문

아래 값을 Firebase Console → 프로젝트 설정 → 내 앱 → Android 앱 (com.andy.howareyou) → SHA 인증서 지문에 붙여넣으세요.

## Debug SHA-1 (개발/테스트용)
89:C1:47:04:B6:F7:A4:17:AC:A1:B9:18:19:5F:42:41:82:0C:D7:B3

## Release SHA-1 (스토어 배포용)
아래 스크립트 실행 후 출력된 값을 복사해 Firebase에 추가하세요:
```
.\get-release-sha1.ps1
```

등록 방법:
1. Firebase Console 접속
2. 프로젝트 설정 (톱니바퀴) → 내 앱
3. Android 앱 (com.andy.howareyou) 선택
4. "디지털 지문 추가" 클릭
5. SHA-1 선택 후 위 값 붙여넣기
6. "저장" 클릭
