# 결제·구독 QA 체크리스트

> 배포 전 필수 검증 항목

---

## 1. 신규 구매

| 항목 | 확인 |
|------|:----:|
| 월간 구독 성공 → Firestore `subscriptionStatus: 'premium'` 반영 | ☐ |
| 연간 구독 성공 → Firestore `subscriptionExpiry` 정확히 설정 | ☐ |
| 2명 제한 해제 (3명 이상 보호대상자 추가 가능) | ☐ |
| 6/9/12h 무이동 알림 수신 | ☐ |

---

## 2. 구매 취소/만료

| 항목 | 확인 |
|------|:----:|
| 구독 취소 후 만료일 도달 → `syncPremiumExpiry`로 `inactive` 처리 | ☐ |
| 만료 후 2명 제한 복원 | ☐ |
| 만료 후 6/9/12h 알림 미수신 | ☐ |

---

## 3. 구매 복원

| 항목 | 확인 |
|------|:----:|
| 앱 삭제 후 재설치 | ☐ |
| "구매 복원" 버튼 탭 | ☐ |
| 프리미엄 재활성화 확인 | ☐ |

---

## 4. 무료 보호자

| 항목 | 확인 |
|------|:----:|
| 무료 보호자에게 6/9/12h 알림 **절대** 발송되지 않음 | ☐ |
| 같은 subject에 무료+프리미엄 보호자 2명 → 프리미엄만 수신 | ☐ |

---

## 5. 보안

| 항목 | 확인 |
|------|:----:|
| 클라이언트에서 `subscriptionStatus`/`subscriptionExpiry` 쓰기 불가 | ☐ |
| `GuardianService.getSubscriptionState()`로 읽기만 수행 | ☐ |
