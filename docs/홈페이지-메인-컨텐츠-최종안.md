## 2. HERO 섹션

### 메인 문구 (A/B 테스트 후 선택)

| 버전 | 문구 | 특징 |
|------|------|------|
| **A** | 혼자 사는 가족, 오늘은 괜찮을까요? | 시간성, 전환 강화 |
| **B** | 연락은 줄었지만, 걱정은 남았습니다. | 감정 압축, 모바일 친화 |
| **C** | 혼자 사는 시대. / 안부는 남겨야 합니다. | 브랜드 확장성 |

### 서브 문구

**매일 전화는 못 해도, 하루 한 번 안부는 전할 수 있습니다.**

### 보조 문구 (선택)

**하루 3초, 최소한의 연결**

### 배지

**1개월 무료** (작게)

### CTA

**[지금 시작하기]**

### 긴급성 (베타용)

**베타 1기 모집 중 · 소수 인원으로 먼저 시작합니다.**

- 실제 제한이 있을 경우 "선착순 100명" 사용 가능
- 과도한 희소성은 신뢰에 부정적

## 3. 상황 카드 섹션 (3개 유지)

### 카드 1 — 혼자 사시는 부모님

- **제목**: 혼자 계신 부모님
- **설명**: "괜찮다"는 말 한 마디면 충분합니다.
- **타겟**: 자식이 부모를 걱정하는 사람

### 카드 2 — 타지에서 생활하는 자녀

- **제목**: 타지에서 생활하는 자녀
- **설명**: 시험·야근이 있어도, 부모는 그날 하루가 괜찮았는지 알고 싶어 합니다.
- **타겟**: 부모가 자녀를 걱정하는 사람

### 카드 3 — 혼자 사는 나 (싱글 확장 포함)

- **제목**: 혼자 사는 나, 또는 바빠서 연락이 줄어든 우리 가족
- **설명**: 나를 한 번이라도 체크해줄 사람만 있어도 고립감은 줄어듭니다.
- **강조 문구**: **나도 누군가의 보호대상자가 될 수 있습니다.**
- **타겟**: 1인 가구, 맞벌이·바쁜 가족

## 4. 중간 CTA 섹션

**위치**: 상황 카드 바로 아래 (감정 고조 직후)

### 버튼 문구 (A/B 테스트)

| 옵션 | 문구 |
|------|------|
| A | 우리 가족 연결하기 |
| B | 지금 안부 남기기 |
| C | 우리 가족 시작하기 |
| D | 하루 3초 시작하기 |

- "연결하기"는 추상적, "시작하기"가 클릭 친화적
- 추천: **[우리 가족 시작하기]** 또는 **[지금 안부 남기기]**

---

## 5. 숫자(문제 공감) 섹션

**목적**: 숫자로 현황 제시 → 감정으로 마무리

### 구조

1. **혼자 사는 노인 350만 명.**
2. **1인 가구 700만 이상.**
3. **우리는 점점 따로 살고 있습니다.**
4. **하지만 걱정은 그대로입니다.**

- 숫자 후 반드시 감정 문장으로 이어가기
- 출처 확인 후 숫자 사용

---

## 6. 해결(서비스 소개) 섹션

### 제목

**하루 3초면 충분합니다.**

### 3단계 흐름 (간결하게)

1. **오늘 상태 선택**
2. **자동 전달**
3. **무응답 시 알림**

### 마무리 문구 (GPT 수정 반영)

| 옵션 | 문구 |
|------|------|
| A | 완전한 침묵이 되지 않도록 하는 작은 장치입니다. |
| B | 연락이 끊기는 순간을 막는 작은 연결입니다. |

- "장치"보다 "연결"이 더 따뜻함
- 추천: **연락이 끊기는 순간을 막는 작은 연결입니다.**

## 7. 신뢰·차별점 섹션

### 핵심 문구

**감시가 아닙니다. 안심입니다.**

### 구체 항목

- 위치를 추적하지 않습니다.
- 평가·점수화를 하지 않습니다.
- 오늘 기록했는지, 안 했는지만 전달됩니다.

## 8. 가격 섹션

### 배치

- HERO: "1개월 무료" 배지만
- 상세: 해결·신뢰 섹션 이후

### 문구

- **하루 33원으로, 서로의 안부를 확인하세요.**
- 1개월 무료 체험
- 월 1,000원 (연 12,000원)
- 보호자만 결제, 보호대상자는 무료

- "월 1,000원"보다 "하루 33원"이 심리적 부담 완화에 유리
