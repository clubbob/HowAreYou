# Android 에뮬레이터 한글 입력 설정 가이드

## 문제 상황
PC에서 Android 에뮬레이터에 한글을 입력할 수 없는 경우, 에뮬레이터에 한글 키보드를 추가해야 합니다.

## 해결 방법

### 방법 1: 에뮬레이터 설정에서 한글 키보드 추가 (권장)

#### 1단계: 에뮬레이터에서 설정 앱 열기
1. 에뮬레이터가 실행 중인 상태에서
2. 앱 목록에서 **설정(Settings)** 앱을 찾아 실행합니다

#### 2단계: 언어 및 입력 방법 설정
1. 설정 화면에서 **시스템(System)** 또는 **일반 관리(General management)** 선택
2. **언어 및 입력(Languages & input)** 또는 **언어(Languages)** 선택
3. **가상 키보드(Virtual keyboard)** 또는 **화면 키보드(On-screen keyboard)** 선택
4. **키보드 관리(Manage keyboards)** 또는 **키보드 추가(Add keyboard)** 선택

#### 3단계: 한글 키보드 추가
1. 사용 가능한 키보드 목록에서 **Gboard** 또는 **한국어(삼성 키보드)** 선택
2. 또는 **키보드 추가(Add keyboard)** → **한국어(Korean)** 선택
3. **한글** 또는 **2벌식** 키보드 레이아웃 선택

#### 4단계: 기본 키보드로 설정
1. 입력 필드를 탭하면 키보드 선택 화면이 나타납니다
2. **한글 키보드**를 선택합니다
3. 또는 설정에서 **기본 키보드(Default keyboard)**를 한글로 변경합니다

---

### 방법 2: Gboard 앱 설치 및 설정

#### 1단계: Gboard 설치
1. 에뮬레이터에서 **Play 스토어** 앱 실행
2. "Gboard" 검색
3. **Google LLC**의 **Gboard** 설치

#### 2단계: Gboard에서 한글 활성화
1. 설정 → 언어 및 입력 → 가상 키보드 → Gboard
2. **언어(Languages)** 선택
3. **언어 추가(Add language)** → **한국어(Korean)** 선택
4. **2벌식** 또는 **천지인** 레이아웃 선택

#### 3단계: 기본 키보드로 설정
1. 설정 → 언어 및 입력 → 기본 키보드
2. **Gboard** 선택

---

### 방법 3: 에뮬레이터 재시작 후 확인

키보드를 추가한 후:
1. 에뮬레이터를 **완전히 종료**합니다
2. 에뮬레이터를 다시 시작합니다
3. 앱에서 입력 필드를 탭하면 한글 키보드가 나타나야 합니다

---

### 방법 4: PC 키보드 직접 입력 (임시 해결책)

에뮬레이터에서 한글 키보드를 활성화하기 전까지, PC 키보드로 직접 입력할 수 있습니다:

1. 에뮬레이터의 입력 필드를 클릭합니다
2. PC 키보드에서 한글을 입력합니다
3. 일부 에뮬레이터에서는 자동으로 한글이 입력됩니다

**주의:** 이 방법은 에뮬레이터 설정에 따라 작동하지 않을 수 있습니다.

---

## 빠른 확인 방법

### 한글 키보드가 활성화되었는지 확인:
1. 앱의 입력 필드를 탭합니다
2. 키보드가 나타나면:
   - **영어 QWERTY** 키보드 → 한글 키보드 추가 필요
   - **한글 2벌식** 키보드 → 정상 작동 중 ✅

---

## 문제 해결

### 한글 키보드가 나타나지 않는 경우:

1. **에뮬레이터 재시작**
   ```powershell
   # 에뮬레이터 완전 종료 후 다시 시작
   ```

2. **앱 재시작**
   - 앱을 완전히 종료하고 다시 실행

3. **키보드 설정 재확인**
   - 설정 → 언어 및 입력 → 가상 키보드
   - 한글 키보드가 목록에 있는지 확인
   - 활성화되어 있는지 확인

4. **다른 키보드 앱 시도**
   - Gboard 대신 다른 한글 키보드 앱 설치 시도

## Pixel 6 / Pixel 7 에뮬레이터별 설정

### Pixel 6 (emulator-5554)
1. 설정 앱 실행
2. 시스템 → 언어 및 입력 → 가상 키보드
3. Gboard → 언어 → 한국어 추가

### Pixel 7 (emulator-5556)
1. 설정 앱 실행
2. 시스템 → 언어 및 입력 → 가상 키보드
3. Gboard → 언어 → 한국어 추가

**참고:** 두 에뮬레이터 모두 동일한 방법으로 설정할 수 있습니다.

## 예상 결과

설정 완료 후:
- ✅ 입력 필드를 탭하면 한글 키보드가 나타납니다
- ✅ PC에서 한글을 입력할 수 있습니다
- ✅ "보호자 이름" 입력 필드에 한글이 정상적으로 입력됩니다

## 추가 팁

### 키보드 전환 단축키
- 에뮬레이터에서 **Space + Shift** 또는 **Ctrl + Space**로 키보드 전환 가능
- 또는 키보드 하단의 **지구본 아이콘**을 탭하여 키보드 전환

### 한영 전환
- 한글 키보드에서 **Space + 한/영** 키로 전환
- 또는 키보드의 **한/영** 버튼 탭
