## 1. 구독 상품 생성

### 1.1 상품 ID

| 상품 ID | 설명 | 청구 주기 |
|---------|------|-----------|
| `premium_monthly` | 월간 구독 | 매월 |
| `premium_yearly` | 연간 구독 | 매년 |

### 1.2 Play Console 경로

1. **수익 창출** → **제품** → **구독**
2. **구독 만들기** 클릭
3. 상품 ID 입력 (`premium_monthly` 또는 `premium_yearly`)
4. 가격 설정 (원화 KRW 권장)
5. 무료 체험/프로모션 기간 설정 (선택)

### 1.3 체크리스트

- [ ] `premium_monthly` 구독 생성 완료
- [ ] `premium_yearly` 구독 생성 완료
- [ ] 각 구독이 **활성** 상태
- [ ] 테스트 계정 등록 (수익 창출 → 설정 → 라이선스 테스트)

## 2. 서비스 계정 (서버 검증용)

### 2.1 Google Cloud Console

1. [Google Cloud Console](https://console.cloud.google.com) → 프로젝트 선택
2. **IAM 및 관리자** → **서비스 계정** → **서비스 계정 만들기**
3. 이름: `play-subscription-verifier` (예시)
4. **키** 탭 → **키 추가** → **새 키 만들기** → JSON
5. JSON 키 파일 다운로드 (리포지토리에 포함 금지)

### 2.2 Play Console 연결

1. **Play Console** → **설정** → **API 액세스**
2. **연결** → Google Cloud 프로젝트 연결
3. **서비스 계정 연결** → 위에서 만든 서비스 계정 선택
4. 권한: **재무 데이터, 주문 및 구독 취소** (또는 **보기 전용**)

### 2.3 Firebase Functions 환경변수

```bash
# 방법 A: Firebase config (JSON 문자열, 따옴표 이스케이프 주의)
firebase functions:config:set play.credentials="$(cat path/to/service-account.json | jq -c .)"

# 방법 B: 환경변수 (배포 시)
# PLAY_SERVICE_ACCOUNT_JSON='{"type":"service_account",...}' firebase deploy --only functions
```
