# HowAreYou (오늘 어때) - 전체 구축내용 정리

> 2026년 2월 기준 · **무료 전환 버전** (구독/결제/베타 제거)

---

## 1. 서비스 개요

**서비스명**: 오늘 어때?  
**핵심 가치**: 떨어져 사는 가족을 위한 안심 루틴 앱  
**타겟**: 혼자 사는 부모님이 걱정될 때, 혼자 살며 누군가와 안부를 나누고 싶을 때, 떨어져 있는 가족의 안부가 궁금할 때

### 한 줄 소개
> 전화 대신, 하루 한 번 3초 안부. 버튼 한 번이면 오늘의 안부는 끝납니다.

## 4. 관리자 웹 (`/admin`)

### 4.1 인증

- 로그인: 아이디/비밀번호 (ADMIN_USERNAME, ADMIN_PASSWORD)
- 세션 쿠키 기반
- 모든 관리자 페이지 접근 시 인증 확인

### 4.2 대시보드

- 통계: 회원 수, 미답변 문의 수, 공지 수, 서비스 개선 피드백 수
- 빠른 링크: 회원관리, 1:1 문의, 서비스 개선, 공지사항

### 4.3 회원관리

- 전화번호, 이름, 가입일시, 역할(보호대상자/보호자/둘 다)
- 연결된 보호자/보호대상자 전화번호
- 번호·이름 검색, 역할 필터, 정렬, 새로고침

### 4.4 공지사항

- 등록/수정/삭제, 상단 고정
- 목록: 제목만 표시, 클릭 시 내용 펼침

### 4.5 1:1 문의

- 목록 조회, 상태 필터(미답변/답변완료), 기간 필터
- 답변 등록, 문의 삭제

### 4.7 서비스 개선 피드백

- 앱에서 제출한 만족도, 불편했던 점, 개선 아이디어, 계속 사용 의향
- 기간 필터: 전체 / 최근 7일 / 최근 30일

## 5. 모바일 앱 (Flutter)

### 5.1 인증

- **휴대폰 OTP** (Firebase Auth) - Google 로그인 미사용
- 약관 동의
- 최초 가입 시 환영 다이얼로그

### 5.2 역할

- **보호대상자**: 안부 기록, 보호자 관리, 내 상태 보기
- **보호자**: 보호대상자 목록, 대시보드, 연속 기록·7일 이력 확인

### 5.3 보호대상자 기능

- 오늘 상태 기록 (5가지 이모지 + 메모)
- 보호자 목록, 보호자 추가(전화번호/초대 링크)
- 7일 이력 차트, 연속 기록(스트릭)
- **프리미엄**: 핸드폰 이동 감지 (MovementDetectionService, Activity Recognition)
- 설정, 공지사항, 1:1 문의, 서비스 개선 피드백
- 당일 미기록 시 19:00 리마인드 알림

### 5.4 보호자 기능

- 보호대상자 등록: 무료 2명 / 프리미엄 무제한
- 보호대상자별 오늘 기록 여부, 7일 이력, 연속 기록
- 보호대상자 추가/제거
- 기록 시 실시간 알림, 3일 연속 무응답 시 알림
- **프리미엄**: 6h/9h/12h 핸드폰 무활동 알림 (lastMovementAt 기준)
- 공지사항, 1:1 문의

## 6. API 및 백엔드

### 6.1 홈페이지 API (Next.js)

| 경로 | 메서드 | 설명 |
|------|--------|------|
| `/api/inquiry` | POST | 문의 등록 |
| `/api/inquiry/check` | GET | 문의 확인 |
| `/api/inquiry/delete` | POST | 문의 삭제 |
| `/api/admin/login` | POST | 관리자 로그인 |
| `/api/admin/logout` | POST | 로그아웃 |
| `/api/admin/auth` | GET | 세션 확인 |
| `/api/admin/stats` | GET | 통계 |
| `/api/admin/users` | GET | 회원 목록 |
| `/api/admin/announcements` | GET, POST | 공지 조회/등록 |
| `/api/admin/announcements/[id]` | PATCH, DELETE | 공지 수정/삭제 |
| `/api/admin/inquiries` | GET | 문의 목록 |
| `/api/admin/inquiries/[id]` | DELETE | 문의 삭제 |
| `/api/admin/inquiries/[id]/reply` | POST | 답변 등록 |
| `/api/admin/service-feedback` | GET | 서비스 개선 피드백 목록 |

### 6.2 Firebase Cloud Functions

| 함수 | 설명 |
|------|------|
| `addGuardianToSubject` | 보호자→보호대상자 연결 (Callable, 서버 레벨 무료 2명/프리미엄 무제한) |
| `addGuardianToSubjectBySubjectInvite` | 보호대상자→보호자 추가 (Callable, 프리미엄 체크) |
| `removeGuardianFromSubject` | 보호자가 자신을 제거 (Callable) |
| `removeGuardianFromSubjectBySubject` | 보호대상자가 보호자 제거 (Callable, guardianSubjectCount 동기화) |
| `processPendingInvitesOnSignup` | 신규 가입 시 대기 초대 자동 연결 |
| `migrateGuardianSubjectCount` | guardianSubjectCount 마이그레이션 (배포 전 1회, ?key=migrate_guardian_count_2026) |
| `onResponseCreated` | 기록 시 보호자에게 알림 발송 |
| `onResponseUpdated` | 기록 수정 시 보호자 알림 |
| `onSubjectUpdated` | lastMovementAt 갱신 시 무활동 알림 단계 리셋 (P0 치명 버그 방지) |
| `sendSubjectReminder` | 매일 19:00 보호대상자 리마인드 |
| `sendPremiumInactivityAlert` | 30분마다 프리미엄 무활동 알림 (6h/9h/12h 단계) |
| `reportAdminFcmOpened` | 관리자 FCM 열람 보고 (Callable) |
