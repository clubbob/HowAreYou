# 에뮬레이터에서 알림 소리 확인 방법

## 1. 에뮬레이터 볼륨 확인

### 방법 1: 에뮬레이터 설정에서 확인
1. 에뮬레이터에서 **설정(Settings)** 앱 열기
2. **소리(Sound)** 또는 **알림(Notifications)** 메뉴로 이동
3. **볼륨(Volume)** 확인
   - 알림 볼륨이 0이 아닌지 확인
   - 무음 모드가 켜져 있지 않은지 확인

### 방법 2: 하드웨어 키 사용
- 에뮬레이터 오른쪽 사이드바에서 **볼륨 업** 버튼 클릭
- 또는 키보드 단축키 사용 (에뮬레이터마다 다름)

## 2. 호스트 시스템 볼륨 확인

- Windows/Mac/Linux의 시스템 볼륨이 켜져 있는지 확인
- 스피커/헤드폰이 연결되어 있고 볼륨이 켜져 있는지 확인

## 3. 에뮬레이터 알림 채널 설정 확인

앱에서 알림 채널이 제대로 생성되었는지 확인:

1. 에뮬레이터에서 **설정(Settings)** 앱 열기
2. **앱(Apps)** → **지금 어때?** 앱 선택
3. **알림(Notifications)** 메뉴로 이동
4. **보호자 알림** 채널이 있는지 확인
5. 채널 설정에서:
   - **소리(Sound)** 켜져 있는지 확인
   - **중요도(Importance)**가 "높음(High)"인지 확인

## 4. 테스트 방법

### 방법 1: 테스트 알림 발송
앱에서 테스트 알림을 발송하여 소리가 들리는지 확인:

```dart
// 테스트용 코드 (임시로 추가)
await _notifications.show(
  999,
  '테스트 알림',
  '소리가 들리나요?',
  NotificationDetails(
    android: AndroidNotificationDetails(
      'guardian_notifications',
      '보호자 알림',
      importance: Importance.high,
      priority: Priority.high,
      playSound: true,
      enableVibration: true,
    ),
  ),
);
```

### 방법 2: Firebase Console에서 테스트 알림 발송
1. Firebase Console → **Cloud Messaging** 메뉴
2. **새 알림 보내기(Send test message)** 클릭
3. FCM 토큰 입력 (Logcat에서 확인한 토큰)
4. 알림 발송 후 에뮬레이터에서 소리 확인

## 5. 실제 기기에서 테스트 (권장)

에뮬레이터에서 소리가 안 들릴 경우, **실제 Android 기기에서 테스트하는 것이 가장 확실**합니다:

1. USB 디버깅 활성화
2. 기기를 컴퓨터에 연결
3. `flutter run` 실행 시 기기 선택
4. 알림 수신 시 소리 확인

## 6. 문제 해결

### 소리가 안 들릴 때

1. **에뮬레이터 재시작**
   - 에뮬레이터를 완전히 종료하고 다시 시작

2. **앱 재설치**
   ```bash
   flutter clean
   flutter run
   ```

3. **알림 권한 확인**
   - 앱에서 알림 권한이 허용되어 있는지 확인
   - 설정 → 앱 → 지금 어때? → 알림 권한 확인

4. **에뮬레이터 오디오 설정 확인**
   - Android Studio → AVD Manager
   - 에뮬레이터 편집 → Advanced Settings
   - Audio 설정 확인

## 참고

- 에뮬레이터의 오디오 드라이버 문제로 소리가 안 들릴 수 있습니다
- 실제 기기에서 테스트하는 것이 가장 확실한 방법입니다
- 알림은 수신되지만 소리만 안 들릴 수 있습니다 (알림 센터에서 확인 가능)
